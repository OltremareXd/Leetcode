# 561.数组拆分 I

解题思路：刚开始还没看懂题目，想复杂了，后来明白了，就是将数组元素两两组成一对，找到最佳配对方案，使得两两配对的数中较小的数字之和最大。使相差最小的两个数放在一对则能够轻松得到答案，而这个策略只需要将数组进行排序，并将奇数位的数字取出求和即可。

1.将数组进行排序；

2.取出奇数位数组元素；

3.求和，返回该和即为最终答案。

代码如下：

```python
class Solution:
    def arrayPairSum(self, nums: List[int]) -> int:
        return sum(sorted(nums)[::2])
```
